# Scrapping-ML
*Creado por Italo Visconti qn mas*

Script que lista en un archivo Excel todos los productos una busqueda de MercadoLibre, incluyendo su nombre, Imagen (Link), Precio. 

Se usan las librerias de:
- Request HTML
- Pandas
- Pillow
- xlsxwriter
